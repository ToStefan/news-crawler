## News crawler

Flask app for crawling movie news from various websites

If you're using Docker Toolbox you'd need to access http://192.168.99.100:5000


## Commands

Build

```
docker-compose build
```

Run

```
docker-compose up
```

List images

```
docker images
```

List containers

```
docker ps
```

Clear all not running stuff

```
Docker system prune -a
```
